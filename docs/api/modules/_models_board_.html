getBoard

  • getBoard(): EFieldColor[][]
    • Defined in models/board.ts:37

    Returns the game board, which is a two-dimensional array of EFieldColors.

    A position on the board is given by a row and a col entry. row is the first dimension of the array and col the second.

    see

    IPosition

    This board layout never changes. So, it is sufficient to call this function once on application startup and store the board in a constant.

    Returns EFieldColor[][]
