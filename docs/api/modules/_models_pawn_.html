  • getIndexOfPawnInDeadlock(pawns: IPawn[], limits: ILimits): number

moveGenerator

  • moveGenerator(pawns: IPawn[], pawnI: number, limits: ILimits, offset: IPosition): IPosition[]
    • Defined in models/pawn.ts:187

    Returns an array of possible moves starting (not including) at the pawns current postion. It than moves step by step from the start position along the direction given by the offset parameter. It does that as long as the found positions are valid moves and stops once that is no longer the case.

    Parameters

    • pawns: IPawn[]
