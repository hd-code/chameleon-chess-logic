Interface IGameState

This is the main data structure for playing a game of chameleon chess. It represents the current state of the game with all needed information.

It holds the following properties:

  • limits: specify the current size of the game board (see ILimits)
  • pawns: an array with all the pawns that are still in play/alive (see IPawn)
  • player: the player who is currently on turn

All other information about the current game, can be derived from this game state object.
